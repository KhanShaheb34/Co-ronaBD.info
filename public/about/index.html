nCov-19 Interactive Dashboard for Bangladesh

A interactive dashboard with Bangladesh map showing the number of positive cases in every district.

Source Code

This is a open source project. Source code is available in this github repository.

Data Collected From

Covid-19 Bangladesh: District-wise data

JSON time-series of coronavirus cases

Bangladesh Map GeoJSON from geodash.gov.bd

I Used

React to build UI

Leafet and react-leaflet to show the map

Bootstrap to style

MapShaper to compress the map
